# BiliBiliVideoCount
B站视频信息统计 支持AV/BV号 树莓派+OLED屏幕

## 屏幕驱动以及接线
Adafruit_SSD1306
![Image text](https://github.com/LunaroakF/Images/blob/master/RaspberryPiMusicPlayer/Line.jpg)  

## 用法  
将128x64的I2C的OLED屏幕连接到树莓派  
确保驱动正常  
修改BiliBili.py第13~15行的内容  
然后运行
